## **Example 1 – Software Engineer (Backend)**

### Job Description (JD)

```
We are looking for a Software Engineer with strong backend development skills.  
Responsibilities:
- Design and develop scalable APIs and microservices.  
- Work with databases such as PostgreSQL and MongoDB.  
- Collaborate with frontend engineers and product managers.  
- Write clean, efficient, and maintainable code in Node.js or Python.  

Requirements:
- Bachelor’s degree in Computer Science or related field.  
- 2+ years of backend development experience.  
- Strong knowledge of REST APIs, databases, and cloud services (AWS preferred).  
- Good problem-solving and communication skills.  
- Familiarity with version control systems (Git, GitHub).  
```

## **Example 2 – Data Analyst**

### Job Description (JD)

```
We are hiring a Data Analyst to join our growing analytics team.  

Responsibilities:
- Collect, clean, and analyze large datasets.  
- Build dashboards and reports using Power BI or Tableau.  
- Work with SQL and Python to perform data analysis.  
- Collaborate with business stakeholders to derive insights.  

Requirements:
- Bachelor’s degree in Statistics, Mathematics, Computer Science, or related.  
- 1–3 years of experience in data analysis or business intelligence.  
- Strong SQL and Python skills.  
- Experience with data visualization tools (Power BI, Tableau).  
- Knowledge of Excel and statistical methods.  
```
